# CyNotes

Minimalistic Notes app made in React Native

## Install

- clone this repository using `git`

- install dependencies

    ```sh
    npm install
    ```

## Usage

- start development server

    ```sh
    npm run start
    ```

- open [**Expo Go**](<https://expo.dev/go>) app on phone and scan generated QR code
